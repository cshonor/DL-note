## 学习目标

- 理解深度学习基础原理与核心思想
- 掌握前向传播、梯度下降、反向传播等基础算法
- 实现神经网络、CNN、RNN、LSTM 等模型
- 手写简易自动求导框架
- 了解联邦学习等前沿方向
