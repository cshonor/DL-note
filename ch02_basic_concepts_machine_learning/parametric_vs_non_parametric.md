# 参数学习 vs 非参数学习（面向 AI Agent / LLM / RAG）

我用**最通俗、不搞公式、跟你找工作（AI Agent、LLM、RAG）强相关**的方式讲明白：
监督/无监督是按**数据有没有标签**分；
参数/非参数是按**模型学完后，要不要把原始数据带在身上**分。

---

## 一、先一句话抓住本质

### 1. 参数学习（Parametric Learning）
**学完就把知识“压缩”成一组参数，扔掉原始数据。**  
模型变小、变快、固定大小。

### 2. 非参数学习（Non-parametric Learning）
**学完不扔数据，推理时还要回去查原始数据。**  
模型大小不固定，数据越多越“重”。

---

## 二、用大白话对比

### 1）参数学习 = 把书读进脑子里
- 训练：看大量数据 → 总结规律 → 记成**一组参数（权重）**
- 推理：只靠这组权重预测，**不再看原始数据**
- 特点：
  - 模型体积固定（几MB～几GB）
  - 预测快
  - 但**学完就定型，很难随时加新知识**

**典型例子：**
- 线性回归
- 逻辑回归
- 神经网络、CNN、Transformer、LLM 大模型（训练阶段）
- 各种预训练模型

### 2）非参数学习 = 不背书，考试时翻书/查笔记
- 训练：基本不“压缩知识”，更多是在建索引/存样本
- 推理：来了新问题 → **去原始数据里找相似例子** → 再做判断
- 特点：
  - 可以**随时加新数据**，不用重训
  - 模型大小随数据增长
  - 推理通常慢一点（取决于检索与索引质量）

**典型例子：**
- KNN（找最相似的 K 个样本）
- **RAG（检索增强生成）**
- 向量数据库检索（如 Chroma、Pinecone 等）

---

## 三、跟你现在学的东西对应上（最重要）

你现在学 **LLM + Agent + RAG**，刚好对应：

### LLM 本身 = 参数学习
- 训练时把世界知识压缩进 **Transformer 参数**
- 对话、生成、理解，全靠参数
- 不能随便加新知识：要更新通常需要微调/继续训练

### RAG = 非参数学习
- 不改动模型参数
- 你上传文档，它存起来
- 提问时去**检索相关段落**，再塞给模型作为上下文
- 这就是典型的**非参数风格**：用数据直接增强回答

### Agent 系统 = 参数 + 非参数 混合
- 大脑（LLM）：参数学习
- 工具查询、数据库、RAG：非参数学习

业界主流落地路线常见组合是：  
**参数模型做理解 + 非参数检索做知识更新**（也就是 **LLM + RAG + Agent**）

---

## 四、极简总结（方便面试复述）
- **监督/无监督**：看数据有没有标签（学习目标）
- **参数/非参数**：看模型要不要背着原始数据跑（存储方式）

- **参数学习**：学完扔掉数据，只带权重 → 固定大小、快、难更新
- **非参数学习**：一直带着数据，用时去查 → 灵活、可随时加新知识
