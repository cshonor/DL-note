## 3.7 多个输入：完整可运行代码

### 一、本节核心定位

承接 3.6 向量点积与前向传播逻辑，给出**原生 Python 手写版 + NumPy 高性能版**两套完整神经网络代码，二者运算逻辑一致，在本书示例数据下最终输出均为 `0.98`。

---

### 二、版本 1：Python 原生基础代码（手动实现点积）

完全不依赖第三方库，用循环实现加权和（向量点积），便于看清底层运算。

```python
def w_sum(a, b):
    assert len(a) == len(b)
    output = 0
    for i in range(len(a)):
        output += a[i] * b[i]
    return output


weights = [0.1, 0.2, 0]


def neural_network(input_vec, weights):
    pred = w_sum(input_vec, weights)
    return pred


toes = [8.5, 9.5, 9.9, 9.0]
wlrec = [0.65, 0.8, 0.8, 0.9]
nfans = [1.2, 1.3, 0.5, 1.0]

input_vec = [toes[0], wlrec[0], nfans[0]]

pred = neural_network(input_vec, weights)
print(pred)
```

### 三、版本 2：NumPy 极简高性能代码

NumPy（Numerical Python）面向向量/矩阵运算，底层实现经过优化；工程里大量数值代码会采用与 NumPy 类似的向量化写法。

```python
import numpy as np

weights = np.array([0.1, 0.2, 0])


def neural_network(input_vec, weights):
    pred = input_vec.dot(weights)
    return pred


toes = np.array([8.5, 9.5, 9.9, 9.0])
wlrec = np.array([0.65, 0.8, 0.8, 0.9])
nfans = np.array([1.2, 1.3, 0.5, 1.0])

input_vec = np.array([toes[0], wlrec[0], nfans[0]])

pred = neural_network(input_vec, weights)
print(pred)
```

---

### 四、两种写法核心对比

1. **运算本质相同**  
   手写 `w_sum` 与 NumPy 的 `.dot()`（一维情形下）都是**向量点积 / 加权和**，对应多输入单层网络的一次前向传播；在相同数据与权重下结果一致。

2. **写法差异**
   - **原生 Python**：显式循环，适合对照公式理解每一步。
   - **NumPy**：向量化 API，代码短、通常更快，是数值计算里的常见习惯写法。

3. **维度要求不变**  
   输入向量与权重向量**长度必须相等**，点积才有定义。

---

### 五、章节核心总结

1. 多输入神经网络的一次前向传播，在本章线性设定下就是**向量点积（加权和）**。
2. 手写循环帮助建立直觉，NumPy（及后续框架中的张量运算）帮助落地到高效实现。
3. 更复杂的深度学习框架在概念上仍建立在这种**线性组合 + 后续层/激活**的堆叠之上（具体 API 会进一步封装）。
