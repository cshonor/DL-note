# 《深度学习图解》第3章 · 3.8 单输入多输出与逐元素乘法

## 3.8 多个输出：一份输入，多路预测

### 一、与前几节的对比

| 类型 | 输入 | 输出 | 核心运算 |
|------|------|------|----------|
| 多输入单输出 | 向量 | 标量 | 点积 / 加权和（再求和） |
| **单输入多输出** | **标量** | **向量** | **标量 × 权重向量的逐元素乘法**（不求和成一项） |

书中场景：只用当前**胜负记录**一个数，同时要预测受伤比例、是否胜利、悲伤程度等多类结果——一路输入、多路权重、得到**多个预测值组成的列表（向量）**。

### 三、数学与直觉

设标量输入为 \(x\)，权重向量为 \([w_0, w_1, w_2]\)，则输出向量为：

$$
[x \cdot w_0,\; x \cdot w_1,\; x \cdot w_2]
$$

与「多输入单输出」里把所有 \(x_i w_i\) **再加起来**不同，这里**每一项各算各的预测**，互不合并成一个标量。

---

### 四、完整可运行代码

```python
def ele_mul(number, vector):
    output = [0] * len(vector)
    for i in range(len(vector)):
        output[i] = number * vector[i]
    return output


weights = [0.3, 0.2, 0.9]


def neural_network(input_scalar, weights):
    pred = ele_mul(input_scalar, weights)
    return pred


wlrec = [0.65, 0.8, 0.8, 0.9]
input_scalar = wlrec[0]

pred = neural_network(input_scalar, weights)
print(pred)  # [0.195, 0.13, 0.585]
```

---

### 五、小结

1. **多输出** = 标量输入与权重向量**逐元素相乘**，得到与权重等长的预测向量。  
2. 实现上对应 **`ele_mul`**（书中命名），再包一层 **`neural_network`** 即可。  
3. 与多输入单输出的 **`w_sum` / 点积** 成对出现，合起来覆盖「向量与标量、向量与向量」两种最基本组合方式。
