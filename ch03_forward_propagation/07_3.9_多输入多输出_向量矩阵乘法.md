## 3.9 多个输入且多个输出：权重矩阵与 `vect_mat_mul`

### 一、本节定位

在 **多输入单输出**（点积求和）与 **单输入多输出**（标量与向量逐元素乘）之后，把两者合起来：**输入仍是特征向量，输出是多个预测值**，每一行权重与输入做一次点积，得到对应输出的标量。整体即 **输入行向量 × 权重矩阵 → 输出行向量**（与线性代数里矩阵乘法的这一形态一致）。

### 三、权重矩阵怎么摆

书中约定（与图一致）：**矩阵的每一行对应一个输出神经元**，**每一列对应一个输入特征**（脚趾、胜负记录、粉丝数）。

```text
#           toes   win   fans
weights = [[ 0.1,  0.1, -0.3],   # 是否受伤
           [ 0.1,  0.2,  0.0],   # 是否胜利
           [ 0.0,  1.3,  0.1]]   # 是否难过
```

输入向量 `input_vec = [8.5, 0.65, 1.2]` 须与**每一行的列数**一致，才能与各行分别做点积。

---

### 四、分路计算（与图一致）

记 `x = [8.5, 0.65, 1.2]`。

| 输出     | 加权和 |
|----------|--------|
| 是否受伤 | 8.5×0.1 + 0.65×0.1 + 1.2×(−0.3) = 0.555 |
| 是否胜利 | 8.5×0.1 + 0.65×0.2 + 1.2×0 = 0.98 |
| 是否难过 | 8.5×0 + 0.65×1.3 + 1.2×0.1 = 0.965 |

---

### 五、完整可运行代码

```python
def w_sum(a, b):
    assert len(a) == len(b)
    output = 0
    for i in range(len(a)):
        output += a[i] * b[i]
    return output


def vect_mat_mul(vect, matrix):
    assert len(vect) == len(matrix[0])
    output = []
    for row in matrix:
        output.append(w_sum(vect, row))
    return output


def neural_network(input_vec, weights_matrix):
    pred = vect_mat_mul(input_vec, weights_matrix)
    return pred


weights = [
    [0.1, 0.1, -0.3],
    [0.1, 0.2, 0.0],
    [0.0, 1.3, 0.1],
]

toes = [8.5, 9.5, 9.9, 9.0]
wlrec = [0.65, 0.8, 0.8, 0.9]
nfans = [1.2, 1.3, 0.5, 1.0]

input_vec = [toes[0], wlrec[0], nfans[0]]

pred = neural_network(input_vec, weights)
print(pred)  # [0.555, 0.98, 0.965]
```

---

### 六、与前几种形态对照

| 形态 | 运算要点 |
|------|-----------|
| 多输入单输出 | 一行权重与输入点积，得到一个数 |
| 单输入多输出 | 标量与权重向量逐元素相乘，得到长度与权重相同的向量 |
| **多输入多输出** | **多行权重，每行与输入点积一次，得到长度等于行数的向量** |

---

### 七、小结

全连接一层（无激活）的前向传播，在「多输入多输出」情形下就是 **对权重矩阵的每一行做一次 `w_sum`**；`vect_mat_mul` 把这件事写清楚后，整层预测就是一行 `neural_network` 调用。
