## 误差：衡量预测偏离目标的程度

图中流程：**输入 × 权重旋钮 → 预测**，再与**目标值**比较得到误差。这里先用**均方误差**的一种最简单形式：把「预测 − 目标」平方，作为标量误差（恒为非负，且放大大误差）。

![输入、权重、预测与误差（书中示意）](img/1.png)

---

## 与书中一致的代码

在 Jupyter Notebook 中运行应打印 **`0.3025`**（预测为 `0.5 × 0.5 = 0.25`，与目标 `0.8` 的差的平方）。

```python
knob_weight = 0.5
x = 0.5  # 书中变量名 input，此处改用 x 以免覆盖内置 input
goal_pred = 0.8

pred = x * knob_weight
error = (pred - goal_pred) ** 2

print(error)
```

**说明：** 对「原始误差」`pred - goal_pred` 做平方，一方面把符号统一成**非负**（负的「差多少」与正的同样惩罚量级时更直观），另一方面大偏差会被平方**放大**，便于后续用梯度下降拉向目标。

---

## 数值核对

- 预测：`0.5 × 0.5 = 0.25`
- 原始误差：`0.25 − 0.8 = −0.55`
- 平方误差：`(−0.55)² = 0.3025`
