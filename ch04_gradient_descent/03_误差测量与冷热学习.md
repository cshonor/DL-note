## 一、4.5 为什么需要测量误差？

### 1. 核心作用

把「网络好不好」压成一个标量 **Loss**，训练就变成：**调权重，让 Loss 下降**。比抽象地要求「误差立刻变零」更好落地；优化目标明确，才好设计更新规则。

### 2. 平方误差 vs 绝对误差（直观优先级）

- **平方（均方）误差**：大偏差平方后涨得更快，训练会更用力纠大错；小偏差平方后影响相对小。  
- **绝对误差**：|e| 上每 1 单位错罚 1 单位，**不按平方放大**大错的「优先级」。

### 3. 为什么要「恒正」的损失？

若只用有正有负的「原始差」做平均，**正负会抵消**：例如 +1000 与 −1000 平均为 0，但两次预测都很差。平方（或其它只依赖大小的形式）让贡献**不可互相抵消**，避免被平均数骗过去。

## 二、4.6 冷热学习（最朴素的试探更新）

### 1. 是什么

**冷热学习**：在当前权重附近，分别**试探**「略增大」「略减小」两个方向，看哪个方向的**平方误差更小**，就把权重往那边挪一小步；循环多次，误差逐步下降。

### 2. 单轮可概括为 5 步

1. 用当前 **weight** 算预测与 **Loss**。  
2. 试探 **weight + step**（热），算新 **Loss**。  
3. 试探 **weight − step**（冷），算新 **Loss**。  
4. 比较两步的 **Loss**，取更小的一侧更新 **weight**。  
5. 重复足够多轮（书中常用上千次量级演示）。

### 3. 本质

在**一维权重**上做的**局部搜索**：没有显式求导，但体现「**看误差往哪边走**」这一训练思想；可看作梯度下降的非常粗糙的**原型**（真正梯度法用导数指明方向，不必每步试两次前向）。

### 书中示意图：权重大/小试探与择优更新

![三步：更大权重测误差 → 更小权重测误差 → 比较后更新（书中示意）](img/3.png)

图中各格 **`lr`** 与 **0.1 / 0.01** 可能与下一节统一代码的 **`step_amount`** 不完全一致，以你运行时的**单一学习率**为准；逻辑始终是：**两次试探的平方误差谁更小，权重就往哪边挪一步**。

## 三、4.7 完整可运行代码（已校对）

约定与 **`02`** 一致：**Loss = (预测 − 目标)²**，不要用 **(目标 − 预测)²** 混用符号（平方后数值一样，但和全书其它推导写法则统一为 **(pred − goal)²**）。

```python
weight = 0.5
x = 0.5  # 书中名 input，此处用 x
goal_prediction = 0.8
step_amount = 0.001

for iteration in range(1101):
    prediction = x * weight
    error = (prediction - goal_prediction) ** 2

    up_prediction = x * (weight + step_amount)
    up_error = (up_prediction - goal_prediction) ** 2

    down_prediction = x * (weight - step_amount)
    down_error = (down_prediction - goal_prediction) ** 2

    if down_error < up_error:
        weight -= step_amount
    elif down_error > up_error:
        weight += step_amount
    # 二者相等时可不更新
```

### 运行现象（与书一致）

- 起步：预测 **0.25**，Loss **0.3025**。  
- 多轮后：预测可逼近 **0.8**，Loss 趋近 **0**（步长、轮数与浮点实现会影响终点精度）。

### 优缺点

| 优点 | 缺点 |
|------|------|
| 逻辑极简，便于建立「误差驱动更新」直觉 | 每步两次额外前向，**极慢**；权重一多组合爆炸，**无法**当实用训练法 |

## 四、带日志的可运行脚本（可选）

每隔 200 轮打印一次，便于观察下降（轮数可按需改小做快速试验）：

```python
weight = 0.5
x = 0.5
goal = 0.8
step_amount = 0.001

for i in range(1101):
    pred = x * weight
    err = (pred - goal) ** 2

    up_e = (x * (weight + step_amount) - goal) ** 2
    dn_e = (x * (weight - step_amount) - goal) ** 2

    if dn_e < up_e:
        weight -= step_amount
    elif dn_e > up_e:
        weight += step_amount

    if i % 200 == 0 or i == 1100:
        print(f"iter={i:4d}  weight={weight:.6f}  pred={x*weight:.6f}  loss={err:.6f}")
```

## 五、全书逻辑串讲：冷热缺陷 → 梯度下降

### 1. 冷热学习两大硬伤

**（1）效率极低**  
每更新 **1 次权重**，除当前点的前向外，还要各做一次 **w+步长**、**w−步长** 的试探预测，等于 **3 次前向** 才走一步；大量重复计算。

**（2）步长固定，难精细收敛**  
步幅 **`step_amount`** 与当前误差大小**脱钩**：误差很大时步长太小则收敛极慢；步长太大则易在最优附近**震荡**；极端时甚至不稳定。

### 2. 梯度下降核心式（与书对齐）

书中最简写法（把「方向+幅度」压进一项里）：

```python
direction_and_amount = (pred - goal_pred) * input
weight = weight - direction_and_amount
```

它对应 **`04`** 里若采用 **`Loss = ½·(pred − y)²`** 时的 **`∂Loss/∂w = (pred − y)·x`**，更新 **`w ← w − η·∂Loss/∂w`**；取 **`η = 1`** 且 **`input` 即 x** 时，就与上面两行代码一致。若损失写成 **`(pred − y)²`（无 ½）**，梯度会多常数 **2**，用更小的 **`η`** 吸收即可（**`04`** 第三节有说明）。

**三个直观作用（书上常概括为：缩放、符号/反转、停更）**

1. **符号（一维里即「往哪边扭 w」）**：**`(pred−y)`** 与 **`x`** 的符号一起决定 **`∂Loss/∂w`** 的正负；更新用 **`w − ηg`**，保证沿**降 Loss** 的方向走（多维时才是严格意义的「方向」）。  
2. **缩放**：误差与 **x** 的绝对值愈大，梯度 **g** 的模愈大，步长自动放大；接近最优时误差小，调节自然变细。  
3. **停更**：**`x = 0`** 时 **`(pred−y)·x = 0`**，该样本不推动 **w** 变化（没有输入信息就不硬改权重）。

### 3. 冷热学习 vs 梯度下降（对照）

| 特性 | 冷热学习 | 梯度下降 |
| :--- | :--- | :--- |
| 方向来源 | 左右试探比 Loss，「猜」下坡 | 对 Loss 求 **∂Loss/∂w**，方向由导数给定 |
| 更新幅度 | 固定 **`step_amount`**，与误差弱相关 | **η** 与梯度 **g** 的模长共同决定步长，可随坡度变化 |
| 前向次数量级 | 每步约 **3** 次预测（当前 + 上 + 下） | 每步通常 **1** 次预测（算梯度再更新） |
| 多权重 | 组合爆炸，难扩展 | 对每个 **w** 各算偏导，可并行，适配网络 |
| 收敛观感 | 易慢、易抖 | 调 **η** 得当则更平滑（仍可能震荡，属另一话题） |

### 4. 一句话串知识点

1. **平方误差**：把误差变成**非负、可优化**的标量 Loss。  
2. **冷热学习**：零基础理解「**看误差、挪权重**」的**玩具**算法。  
3. **梯度下降**：用微积分直接给出**方向与幅度**，是实用训练的主干。  
4. **`(pred−y)·x`**（在 **½** 损失约定下）就是 **w** 的梯度；一步同时回答「往哪走、走多远」。

## 六、用书上的数字手算一步（梯度更新）

与第三节代码一致：**`weight = 0.5`**，**`x = 0.5`**，**`goal_prediction = 0.8`**。

**当前**

- **`pred = x·w = 0.25`**，**`Loss = (0.25 − 0.8)² = 0.3025`**（与 **`02`** 打印一致）。  
- 取 **`Loss = ½(pred − y)²`**，则 **`g = ∂Loss/∂w = (pred − y)·x = (0.25 − 0.8)·0.5 = −0.275`**。  
- 书中极简式 **`w ← w − g`**（等价 **`η=1`** 且 **`w ← w − η·∂Loss/∂w`**）：

**`w_新 = 0.5 − (−0.275) = 0.775`**

**一步之后**

- **`pred_新 = 0.5 × 0.775 = 0.3875`**  
- **`Loss_新 = (0.3875 − 0.8)² ≈ 0.1702`**（比 **0.3025** 明显下降）

若你坚持用 **`Loss = (pred − y)²`（无 ½）**，则 **`g = 2(pred−y)x = −0.55`**；若仍想一步降到相近的新点，需把 **`η`** 取 **0.5**（或其它值），使 **`Δw = −ηg`** 与上式同量级。**`04`** 强调：**差一个常数因子只相当于重标定学习率**，优化对象仍是 Loss。
