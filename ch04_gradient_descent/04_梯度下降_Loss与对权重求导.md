# 《深度学习图解》第4章 · 梯度下降在对谁求导？

> 上接 **`01_预测比较学习与梯度下降链路.md`**；损失形式与书中平方误差见 **`02_误差与均方误差.md`**。本篇只保留**链式法则、拆开算、更新式与符号表**（主干尽量短）。**4.11 极简循环、碗形图、手算与 `input=1.1/2` 书页**、**脚趾四图**、**4.16～4.23 导数/发散** 均在 **`05_书例图解_4.11至4.23.md`**（不用 LaTeX 环境）。

---

## 一、一句话答案（记这个）

**梯度下降，是对损失函数 Loss 求导**——更准确地说，是对**每个待学习参数**求 **Loss 对该参数的偏导数**，例如：

- **∂Loss / ∂w**（权重 w）
- **∂Loss / ∂b**（偏置 b，若模型里有 b）

更新权重时用的就是这一组数（再乘学习率 η，沿**降低 Loss** 的方向走一小步）。

---

## 二、为什么是对 Loss 求导？

目标只有一个：**让 Loss 尽量小**。

**梯度（对某个 w 的偏导）**回答的是：

> 若只把 **w** 动一点点，**Loss** 会**往哪个方向、大约变多少**？

- 偏导数为**正**：w 略增大 → Loss 增大 → 更新时应**减小 w**（沿负梯度走）。  
- 偏导数为**负**：w 略增大 → Loss 减小 → 更新时应**增大 w**。

所以：**「对 Loss 求关于 w 的偏导」**就是更新 **w** 的依据；对 **b** 同理。

---

## 三、完整链路（链式法则）

不是「跳过 Loss 直接对模型瞎求导」，而是：

1. **模型（预测）**：例如单输入线性 **`ŷ = w·x + b`**（书中常从更简单 **`ŷ = w·x`** 起步，b 可后加）。  
2. **损失**：例如均方误差一种常见写法 **`Loss = ½·(ŷ − y_true)²`**（与 `02` 里 **`(pred − goal)²`** 差常数 **½**，只影响梯度整体倍数，**不改变「对谁求导」**的结论）。  
3. **要的量**：**∂Loss / ∂w**、**∂Loss / ∂b**。  
4. **链式**：**∂Loss / ∂w = (∂Loss / ∂ŷ) · (∂ŷ / ∂w)**（中间经过预测 **ŷ**）。

---

## 四、拆开算（以 Loss = ½(ŷ − y_true)²，ŷ = w·x + b 为例）

**① ∂Loss / ∂ŷ**

- **Loss = ½·(ŷ − y_true)²**  
- **∂Loss / ∂ŷ = ŷ − y_true**

**② ∂ŷ / ∂w、∂ŷ / ∂b**

- **ŷ = w·x + b**  
- **∂ŷ / ∂w = x**（此时把 x、b 当常数）  
- **∂ŷ / ∂b = 1**

**③ 合成**

- **∂Loss / ∂w = (ŷ − y_true) · x**  
- **∂Loss / ∂b = (ŷ − y_true) · 1 = ŷ − y_true**

若损失写成 **`Loss = (ŷ − y_true)²`（没有前面的 ½）**，则 **∂Loss / ∂ŷ = 2·(ŷ − y_true)**，进而 **∂Loss / ∂w** 会多一个因子 **2**；更新时把 **η** 调一下即可吸收，**求导对象仍是 Loss**。

---

## 五、「对 Loss 求导」还是「对模型求导」？

- **最终落点**：都是 **Loss 对每个参数的偏导**（**∂Loss/∂w**、**∂Loss/∂b**）。  
- **中间**：要经过 **ŷ = f(x; w, b)**，所以用**链式法则**把 **∂Loss/∂ŷ** 与 **∂ŷ/∂w** 乘起来。  
- **不要记成「对模型求导」与「对 Loss 求导」二选一**；标准说法是：**对参数求导，Loss 在链子的最外一层。**

---

## 六、多个权重时：其余参数当常数

求 **∂Loss / ∂w₂** 时，**w₁、w₃、b** 等在求偏导过程中都**视为常数**；得到的是 **「w₂ 单独动一下，Loss 怎么变」**，即 **w₂ 的专属梯度**。

---

## 七、更新公式（写法对齐）

**w ← w − η · (∂Loss / ∂w)**  

**b ← b − η · (∂Loss / ∂b)**  

**η** 为学习率（步长）。与「导数为正则要减小 w」的叙述一致：减去正梯度，w 减小，Loss 往下降方向走（一维凸情形下的直观；实际曲面更复杂时再谈学习率与收敛）。

---

## 八、极简表

| 问题 | 答案 |
|------|------|
| 梯度下降「对谁」求导？ | **对 Loss**，对每个要学的 **w、b** 求 **∂Loss/∂参数** |
| 为什么出现链式？ | **Loss** 通过 **ŷ** 依赖 **w、b**，要乘 **∂Loss/∂ŷ** 与 **∂ŷ/∂w** 等 |
| 更新依据 | **负梯度方向** × **η** |

## 十、为什么说「方向」？——符号与变化率

对单个标量 **w**，**g = ∂Loss/∂w** 这一个数里同时装了两件事：

- **符号（正 / 负）**：**w** 该**增大还是减小**，才能让 **Loss** **局部**下降（一维里口语说的「方向」几乎就是它）。  
- **绝对值 |g|**：**坡有多陡**、**w 微动一下 Loss 有多敏感**（变化率的大小）。  
- **g = 0**：局部上 **w** 怎么微动 **Loss** 一阶不变（临界点；是否全局最优另说）。

这与 **第二节** 的叙述是同一件事：**正梯度 → 更新要减小 w**；**负梯度 → 更新要增大 w**。这里再钉一句：**「方向」在一维里主要就是导数的正负**，不是另起炉灶的概念。

---

## 十一、再用 pred = w·x 看一遍符号（与第四节一致）

**Loss = ½·(pred − y)²** 时 **g = ∂Loss/∂w = (pred − y)·x**。没有 **½** 时 **g** 多因子 **2**，**符号不变**。

### A. g > 0

**w** 略增 → **Loss** 增；**w** 略减 → **Loss** 减 → 应**减小 w**。**w ← w − ηg** 在 **g>0** 时让 **w** 变小。

### B. g < 0

**w** 略增 → **Loss** 减 → 应**增大 w**。**g<0** 时 **−ηg>0**，**w** 变大。

### C. g = 0

本步可不更新 **w**（一阶上已不敏感）。

---

## 十二、多维与冷热学习

- **多维**：各 **∂Loss/∂wᵢ** 排成向量 **g**；**−g** 是参数空间里**一步最陡下降**的方向；**每个分量的符号**仍表示该坐标该增还是该减。  
- **冷热学习**：试 **w+步长**、**w−步长** 看 **Loss** 谁小，是在**试往哪边挪**；**梯度法**用 **g** 的符号**直接读出**往哪边挪，一般**不必每步试两次**。

---

## 十三、三个具体数（一眼看符号）

仍用 **pred = w·x**，**Loss = ½(pred−y)²**，**g = (pred−y)·x**。

| 情形 | x | y | w | pred | g = (pred−y)·x | 为降 Loss 对 w 的直觉 | w←w−ηg（η>0） |
|------|---|---|---|------|----------------|----------------------|----------------|
| 1 | 0.5 | 0.8 | 0.5 | 0.25 | **−0.275** | 增大 w | w 增大 |
| 2 | 0.5 | 0.1 | 0.5 | 0.25 | **+0.075** | 减小 w | w 减小 |
| 3 | 0.5 | 1.0 | 2.0 | 1.0 | **0** | 一阶不动 | 本步不改 |

第 1 行与 **`02`** 中一步更新 **w** 的直觉一致（pred 低于目标，**g<0**，更新增大 **w**）。
