# 《深度学习图解》第4章 · 平方误差与对数损失：两套数值例题

> 上接 **`04_梯度下降_Loss与对权重求导.md`**（可先浏览 **`06_书例图解_4.11至4.23.md`** 建立几何直觉）。本篇给两个**步步有数**的例子：**平方误差（回归）**、**对数损失 + sigmoid（二分类）**。全文不用 LaTeX 公式环境。

---

## 统一设定（两例共用）

| 量 | 取值 |
|----|------|
| 输入 x | 2 |
| 真实标签 y_true | 见各例（例1 第二段改为 3） |
| 权重 w | 0.5 |
| 偏置 b | 本节先不设，只看 w |
| 学习率 η | 0.1 |

预测写成 **ŷ = w·x**（与书中从单旋钮起步一致）。

---

## 例一：平方误差损失（回归）

损失采用 **Loss = ½·(ŷ − y_true)²**（多出的常数 **½** 只影响梯度整体倍数，与「对谁求导」无关）。

### 情况 A：梯度已为 0（已在当前数据上最优）

1. **预测**：ŷ = w·x = 0.5×2 = **1**  
2. **原始误差**：e = ŷ − y_true = 1 − 1 = **0**  
3. **损失**：Loss = ½·e² = **0**  
4. **梯度**：∂Loss/∂w = (ŷ − y_true)·x = (1−1)×2 = **0**  
5. **更新**：梯度为 0 → **不必改 w**（已使该样本上的平方损失最小）。

### 情况 B：能看到一次完整的 w 更新

改 **y_true = 3**，其余不变（x=2，w=0.5）。

1. **预测**：ŷ = 0.5×2 = **1**  
2. **误差**：e = 1 − 3 = **−2**  
3. **损失**：Loss = ½·(−2)² = **2**  
4. **梯度**：∂Loss/∂w = (ŷ − y_true)·x = (1−3)×2 = **−4**  
5. **更新**：w_new = w − η·(∂Loss/∂w) = 0.5 − 0.1×(−4) = 0.5 + 0.4 = **0.9**

即：**对 Loss 求关于 w 的偏导 → 得到梯度 → 按梯度下降更新 w**。

## 例二：对数损失（二分类，带 sigmoid）

仍取 **x = 2**，**y_true = 1**（正类），**w = 0.5**。先算线性部分 **z = w·x**，再 **ŷ = σ(z)**，再用**二元交叉熵**的一种常见写法（正类标签为 1 时第二项为 0）。

### ① 线性部分

z = 0.5×2 = **1**

### ② Sigmoid 概率

ŷ = σ(z) = 1 / (1 + e^(−z)) = 1 / (1 + e^(−1)) ≈ **0.731**

（e^(−1) ≈ 0.3679，分母约 1.3679。）

### ③ 对数损失（只保留 y_true=1 那一支）

Loss = −ln(ŷ) ≈ −ln(0.731) ≈ **0.313**

### ④ 对 w 的梯度（链式：Loss → ŷ → z → w）

对 **z = w·x**、**ŷ = σ(z)**、**Loss = −[y·ln ŷ + (1−y)·ln(1−ŷ)]** 推到底，有标准结论：

**∂Loss/∂w = (ŷ − y_true)·x**

（与线性均方那一支「形式」相同，是因为 **∂Loss/∂z = ŷ − y** 与 **∂z/∂w = x** 拼起来；**不要**与「Loss 公式相同」混淆。）

代入：∂Loss/∂w ≈ (0.731 − 1)×2 ≈ **−0.538**

### ⑤ 更新 w

w_new = 0.5 − 0.1×(−0.538) = 0.5 + 0.0538 ≈ **0.5538**

---

## 对照总结

| 概念 | 含义 |
|------|------|
| 原始误差（回归里常用） | e = ŷ − y_true |
| 损失函数 | 把 e 或 ŷ 压成一个标量 **Loss**，便于优化 |
| 平方损失（回归） | Loss = ½·(ŷ − y_true)² |
| 对数损失（二分类一例） | 正类：Loss ≈ −ln ŷ（完整式含 (1−y) 项时见教材） |
| 梯度下降求什么 | **∂Loss / ∂w**（对每个 w 各算一遍） |
| 其它参数 | 求 **∂Loss/∂w₁** 时，其余 w、b 当**常数** |
| 更新式 | **w ← w − η·(∂Loss/∂w)** |
