## 一、极简训练循环逐行对应什么

```python
pred = input * weight                  # 1. 前向：由当前 w 得到预测
error = (pred - goal_pred) ** 2        # 2. 比较：平方误差（单样本时常直接叫「误差」打印）
delta = pred - goal_pred               # 3. 原始差（书中常叫 delta / 节点 delta）
weight_delta = delta * input           # 4. 与 ∂Loss/∂w 同型的一项（见下文「½」约定）
weight = weight - weight_delta         # 5. 学习：沿其反方向走一步（等价于 w ← w − η·∂Loss/∂w 的特例）
```

把 **2～5** 放进 `for` 里反复做，就是**最小版「预测 → 损失 → 梯度信息 → 更新」**。实战里往往在 **5** 上乘学习率 **`alpha`**（或 **`η`**），并做批量/正则等，但**骨架**就是这几步。

---

## 二、相对冷热学习，代码里在干什么

1. **不再**左右试探：用 **`delta * input`** 一次给出「该往哪边、动多少」的一阶信息。  
2. 更新式 **`w ← w − weight_delta`** 与 **`w ← w − η · ∂Loss/∂w`** 同族：都是**沿降低 Loss 的方向**动 **w**（符号由 **`delta`** 与 **`input`** 共同决定，见 **`04`** 第十一、十二节）。  
3. 迭代足够多次，**w** 会逼近使 **`pred ≈ goal`** 的值（单样本、严格凸的抛物线情形下，碗底即 **Loss** 最小）。

---

## 三、核心几何：固定 x、y 时，Loss(w) 是「碗」形抛物线

令 **`pred = x·w`**，**`y = goal_pred`**。单样本平方误差（与 **`02`** 打印一致）：

$$
\text{Loss}(w) = (x w - y)^2
$$

例：**`x = 0.5`**，**`y = 0.8`**，则 **`Loss(w) = (0.5w - 0.8)^2`**：横轴 **w**、纵轴 **Loss**，曲线为**开口向上**的抛物线，**全局最低点**在 **`pred = y`** 即 **`0.5w = 0.8`**，**`w^\* = 1.6`** 处（此处 **Loss = 0**）。曲线上每一点对 **w** 的**切线斜率**就是 **dLoss/dw**（一元里即梯度）。

示意图见 **`04`** 第二节后的 **`img/8.png`**（权重–误差与斜率）。

## 四、斜率与代码行的对应（务必对齐「½」）

1. **切线斜率** = **dLoss/dw** = 梯度（标量 **w** 时不再区分「偏导」说法）。  
2. 若采用 **`Loss = ½·(pred − y)²`**（与 **`04`** 第四节同一约定），则  
   **`∂Loss/∂w = (pred − y)·x = delta * input`**，与 **`weight_delta = delta * input`** **完全一致**；此时 **`weight = weight - weight_delta`** 即 **`η = 1`** 的梯度下降一步。  
3. 若 **`Loss = (pred − y)²`（无 ½）**，则 **`∂Loss/∂w = 2·(pred − y)·x`**；仍写 **`weight_delta = delta * input`** 时，相当于**少乘了 2**，可用**更小的步长**或显式写成 **`w -= 2 * delta * input` / `w -= eta * 2*...`** 吸收（**`04`** 第三节：常数只重标定 **η**）。  
4. **符号**：**dLoss/dw > 0** 时，**w** 略增 **Loss** 增 → 更新应**减小 w**；**`< 0`** 则应**增大 w**；**`= 0`** 在一阶上已到**驻点**（单样本抛物线即碗底）。

「变化率里为什么能读方向」：**一元导数的正负**直接回答「**w** 往右微挪时 **Loss** 升还是降」，与「往哪边更新 **w**」一一对应；**绝对值**反映坡陡缓，与步长尺度相关。

---

## 五、与冷热学习划界（概念对照）

|  | 冷热学习 | 梯度下降（本段代码骨架） |
|--|----------|---------------------------|
| 方向 | 试 **w±步长** 比 **Loss** | 用 **dLoss/dw**（或与其成比例的 **`weight_delta`**） |
| 步长 | 试探步与更新步绑死 | 可用 **`alpha`** 单独调；与 **\|dLoss/dw\|** 共同决定一步走多远 |
| 前向次数 | 每步多次预测 | 通常每步一次 **pred** 即可算 **delta** |

---

## 六、一句话

**神经网络学习（入门标量版）**：在 **Loss(w)** 这条「碗形」曲线上，用 **梯度（切线斜率）** 决定往哪挪 **w**，迭代到 **Loss** 足够小；代码里的 **`delta * input`** 就是在实现 **`∂Loss/∂w`**（在 **½** 损失约定下与书中最简写法逐行对齐）。

## 七、手算四轮（`input = 0.5`，`goal_pred = 0.8`，初值 `w = 0.5`）

以下按 **`weight_delta = delta * input`** 且 **`w ← w − weight_delta`**，等价 **`Loss = ½(pred−y)²`**、**`η = 1`** 的一步（与 **`03`** 第六节 **`½`** 手算一步同一约定）。**`error = (pred−y)²`** 仍用**无 ½** 的平方，便于和 **`02`** 的 **0.3025** 对照。

| 轮次 | w | pred = 0.5·w | delta = pred−0.8 | weight_delta = delta·0.5 | 下一轮 w | error = (pred−0.8)² |
|------|---|----------------|-------------------|---------------------------|----------|---------------------|
| 0 | 0.500000 | 0.250000 | −0.550000 | −0.275000 | 0.775000 | 0.302500 |
| 1 | 0.775000 | 0.387500 | −0.412500 | −0.206250 | 0.981250 | 0.170156 |
| 2 | 0.981250 | 0.490625 | −0.309375 | −0.154688 | 1.135938 | 0.095713 |
| 3 | 1.135938 | 0.567969 | −0.232031 | −0.116016 | 1.251953 | 0.053839 |

四轮后 **w ≈ 1.252**，仍在最优 **w\* = 1.6** 左侧，**error** 已从 **0.3025** 持续下降；继续迭代会沿抛物线向碗底靠近。若把一步改成 **`w -= alpha * weight_delta`**（**`alpha = 0.01`**），则每步变化变小，需更多轮次，但**数学对象仍是同一个梯度**乘学习率。
