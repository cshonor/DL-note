# 三种神经网络模型及梯度下降详解

> 目标：基于书中图示，统一梳理三种基础结构的前向计算、损失与梯度更新。  
> 约定：仅讨论正常训练流程，不涉及权重冻结；损失统一用 MSE（平方误差）。

---

## 一、模型 1：多输入 -> 单输出（核心讨论模型）

### 1) 基础结构

- 输入：3 个特征（`x1, x2, x3`）
- 输出：1 个预测（`y_hat`）
- 权重：3 个参数（`w1, w2, w3`），向量形式
- 场景：回归/单一分类打分；也是“权重耦合”最典型的入门模型

### 2) 前向计算

- `y_hat = w1*x1 + w2*x2 + w3*x3`

### 3) 损失计算

- `Loss = (y_hat - y_true)^2`

这里全局只有一个总损失，由三个权重共同决定。

### 4) 梯度下降更新

1. 统一误差项：`error = y_hat - y_true`  
2. 各权重梯度：  
   - `dLoss/dw1 = 2 * error * x1`  
   - `dLoss/dw2 = 2 * error * x2`  
   - `dLoss/dw3 = 2 * error * x3`  
3. 更新：`w_new = w_old - alpha * gradient`（`alpha` 为学习率）

### 5) 核心特点

- 三个权重共享同一个误差信号，彼此强耦合
- 收敛时常表现为多权重协同到低梯度区
- 更容易出现“特征间迁就/捷径拟合”现象

---

## 二、模型 2：单输入 -> 多输出（书中基础分支模型）

### 1) 基础结构

- 输入：1 个特征（`x`）
- 输出：3 个预测（`y_hat1, y_hat2, y_hat3`）
- 权重：3 个标量（`w1, w2, w3`）
- 直观理解：同一个输入，喂给 3 个互相独立的小模型

### 2) 前向计算

- `y_hat1 = w1 * x`  
- `y_hat2 = w2 * x`  
- `y_hat3 = w3 * x`

### 3) 损失计算

- `Loss = (y_hat1 - y_true1)^2 + (y_hat2 - y_true2)^2 + (y_hat3 - y_true3)^2`

总损失是三项之和，但每一项只和对应权重相关。

### 4) 梯度下降更新

1. 分别计算误差：  
   - `error1 = y_hat1 - y_true1`  
   - `error2 = y_hat2 - y_true2`  
   - `error3 = y_hat3 - y_true3`
2. 分别计算梯度：  
   - `dLoss/dw1 = 2 * error1 * x`  
   - `dLoss/dw2 = 2 * error2 * x`  
   - `dLoss/dw3 = 2 * error3 * x`
3. 独立更新：  
   - `w1_new = w1_old - alpha * grad1`  
   - `w2_new = w2_old - alpha * grad2`  
   - `w3_new = w3_old - alpha * grad3`

### 5) 核心特点

- 三路输出、三组权重基本独立
- 某一路更新通常不会直接拖动另外两路
- 相比“多入单出”，耦合更弱、更容易分任务理解

---

## 三、模型 3：多输入 -> 多输出（矩阵化进阶模型）

### 1) 基础结构

- 输入：3 个特征（`x1, x2, x3`）
- 输出：3 个预测（`y_hat1, y_hat2, y_hat3`）
- 权重：`3x3` 矩阵（第 i 行对应第 i 个输出）
- 本质：3 组“多输入 -> 单输出”并排计算，共享同一输入向量

### 2) 前向计算

- `y_hat1 = w11*x1 + w12*x2 + w13*x3`  
- `y_hat2 = w21*x1 + w22*x2 + w23*x3`  
- `y_hat3 = w31*x1 + w32*x2 + w33*x3`

### 3) 损失计算

- `Loss = (y_hat1 - y_true1)^2 + (y_hat2 - y_true2)^2 + (y_hat3 - y_true3)^2`

### 4) 梯度下降更新

1. 先计算每个输出自己的误差  
2. 每一行权重只接收“该行输出”的误差信号  
3. 按同一规则逐元素更新：`w_ij = w_ij - alpha * (dLoss/dw_ij)`

### 5) 核心特点

- 矩阵形式便于向量化与批量计算
- 行内耦合（同一输出对应的多输入权重互相影响）
- 行间相对独立（不同输出对应不同误差信号）

---

## 四、三种模型对比总表

| 模型 | 输入输出 | 权重形式 | 耦合性 | 梯度信号来源 |
| --- | --- | --- | --- | --- |
| 多输入 -> 单输出 | 3 入 1 出 | 向量 | 强耦合 | 共享一个误差 |
| 单输入 -> 多输出 | 1 入 3 出 | 标量组 | 弱耦合/近独立 | 各输出独立误差 |
| 多输入 -> 多输出 | 3 入 3 出 | 矩阵 | 行内耦合、行间相对独立 | 每行对应输出误差 |

---

## 五、一句话总记忆

- 先分清“误差是共享还是分路”  
- 再看“权重是向量、标量组还是矩阵”  
- 最后统一套更新公式：`参数 = 参数 - 学习率 * 梯度`

这三步能快速判断任何基础神经网络的梯度下降逻辑。
