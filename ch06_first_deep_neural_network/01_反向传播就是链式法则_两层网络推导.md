# 反向传播就是链式法则：两层网络一条线讲透

> 只抓一件事：反向传播不是技巧，是链式法则从后往前乘过去。  
> 只讲你卡住的点：前一层梯度怎么从后一层继续推，为什么是乘，不是除。

---

## 一、网络与符号（固定一个最小例子）

结构：`输入 3 个 -> 隐藏 2 个 -> 输出 1 个`

- 输入：`x1, x2, x3`
- 隐藏层输出：`h1, h2`
- 输出预测：`y_hat`
- 标签：`y`
- 损失：`Loss = (y_hat - y)^2`

前向定义：

- `h1 = w11*x1 + w12*x2 + w13*x3`
- `h2 = w21*x1 + w22*x2 + w23*x3`
- `y_hat = v1*h1 + v2*h2`  
  （这里把隐藏层到输出层权重写成 `v1, v2`，避免和前层 `w..` 混淆）

---

## 二、你已经会的：最后一层梯度

先看输出层权重 `v1`：

- `dLoss/dv1 = (dLoss/dy_hat) * (dy_hat/dv1)`
- `dLoss/dy_hat = 2*(y_hat - y)`
- `dy_hat/dv1 = h1`

所以：

- `dLoss/dv1 = 2*(y_hat - y)*h1`

这就是“误差 × 本层输入”。

---

## 三、关键卡点：怎么推到更前面的 `w11`

目标：

- `dLoss/dw11`

链式法则只有一条路径：

- `dLoss/dw11 = (dLoss/dy_hat) * (dy_hat/dh1) * (dh1/dw11)`

三段分别是：

1) `dLoss/dy_hat = 2*(y_hat - y)`  
2) `dy_hat/dh1 = v1`（因为 `y_hat = v1*h1 + v2*h2`）  
3) `dh1/dw11 = x1`（因为 `h1 = w11*x1 + ...`）

乘起来：

- `dLoss/dw11 = 2*(y_hat - y) * v1 * x1`

这就是你问的核心答案：

- **不是除以什么**
- **而是乘以后面那层的权重（这里是 `v1`）**

---

## 四、把规则抽象成通用模板

### 1) 输出层参数

- 梯度 = 误差项 × 该参数输入

### 2) 更前层参数

- 梯度 = 误差项 × 后续路径上的局部导数连乘 × 本层输入

在线性层里，“后续局部导数”常直接表现为后层权重；有激活函数时，还要再乘激活函数导数。

---

## 五、你这次最该记住的一句

**反向传播 = 链式法则。**

从 `Loss` 往回推，每过一层就再乘一个“这一层对下一层的导数”。  
所以深层网络不是新规则，只是“乘法链更长”。

---

## 六、超短口诀

- 输出层：`梯度 = 误差 × 本层输入`
- 前面层：`梯度 = 误差 × 后层导数链 × 本层输入`
- 全程只有乘法链，没有“除法回推”
