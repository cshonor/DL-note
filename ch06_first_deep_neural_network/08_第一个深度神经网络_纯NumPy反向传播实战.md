# 第一个深度神经网络：纯 NumPy 反向传播实战

> 目标：不用任何深度学习框架，手写一个 2 个隐藏层的网络，把“前向 -> 误差 -> 反向 -> 更新”完整跑通。  
> 对应代码：`first_deep_nn_numpy.py`

---

## 一、本节约定

- 网络结构：`4 -> 4 -> 3 -> 1`
- 激活函数：Sigmoid
- 反向传播：链式法则逐层回传
- 默认训练模式：全量批量（BGD）
- 也支持单样本更新（SGD）

## 二、代码（可直接运行）

```python
from first_deep_nn_numpy import run

# 批量梯度下降（默认）
run(mode="batch", epochs=10000, lr=0.1, seed=1)

# 若想看随机梯度下降，改成：
# run(mode="sgd", epochs=10000, lr=0.1, seed=1)
```

脚本内部核心流程：

1. `sigmoid()` 与 `sigmoid_deriv()`  
2. 初始化 `w1, w2, w3`  
3. 前向：`l0 -> l1 -> l2 -> l3`  
4. 反向：`l3_delta -> l2_delta -> l1_delta`  
5. 更新：`w3, w2, w1`

## 三、和第 6 章知识点对应

| 第 6 章主题 | 代码对应 |
| --- | --- |
| 准备数据 | `x, y` 的定义 |
| 矩阵运算 | `@`（矩阵乘法） |

| 反向传播代码化 | 三层 `delta` 的链式回传 |
| 梯度下降更新 | `w += lr * gradient` |
| 批量/随机模式 | `mode="batch"` 与 `mode="sgd"` |

## 四、运行后应该看到什么

- 日志里的误差整体下降（不是每步都单调，但总体下降）
- 最终预测会接近标签 `y`
- 你可以直接验证：
  - 前向是矩阵乘法 + 激活
  - 反向是误差项逐层回传
  - 每条权重都有对应梯度并被更新

## 五、可做的三个小实验

1. **改学习率 `lr`**：看收敛变慢、震荡或发散  
2. **改训练模式**：`batch` 与 `sgd` 对比误差曲线抖动  
3. **改网络宽度**：例如把 `4->3->1` 改成 `4->8->4->1`，观察表达能力变化
