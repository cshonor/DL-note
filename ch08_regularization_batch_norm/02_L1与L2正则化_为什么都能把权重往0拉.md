# L1 与 L2 正则化：为什么都能把权重往 0 拉

## 1) 先看更新公式
梯度下降更新：

`w_j = w_j - eta * dLoss/dw_j`

核心就是看 `dLoss/dw_j` 里，正则项多加了什么。

---

## 2) L2 为什么把权重往 0 拉
若：

`Loss = 误差项 + lambda * sum(w_j^2)`

则：

`dLoss/dw_j = d(误差)/dw_j + 2 * lambda * w_j`

代回更新：

`w_j = w_j - eta * d(误差)/dw_j - eta * 2 * lambda * w_j`

最后一项 `- eta * 2 * lambda * w_j` 永远和 `w_j` 反向：
- `w_j > 0` 时，往下拉；
- `w_j < 0` 时，往上推；

两边都向 `0` 靠，这就是权重衰减（weight decay）直觉。

---

## 3) L1 为什么也把权重往 0 拉
若：

`Loss = 误差项 + lambda * sum(|w_j|)`

则（次梯度视角）：

`dLoss/dw_j = d(误差)/dw_j + lambda * sign(w_j)`

代回更新：

`w_j = w_j - eta * d(误差)/dw_j - eta * lambda * sign(w_j)`

最后一项同样指向 `0`：
- `w_j > 0`，每步减固定量；
- `w_j < 0`，每步加固定量。

---

## 4) L1 与 L2 的关键区别
- **L2**：拉力与 `|w_j|` 成正比（大权重大拉，小权重小拉），更平滑。  
- **L1**：拉力近似固定，不看当前大小，更容易把小权重推到精确 `0`（稀疏化）。

一句话记忆：
- `L2`：温柔收缩，权重普遍变小。  
- `L1`：强硬稀疏，不重要权重更容易归零。

---

## 5) 记号差异说明
有些教材写 L2 为 `(lambda/2) * sum(w_j^2)`，导数会变成 `lambda * w_j`。  
这只是系数写法不同，不改变“往 0 拉”的本质。
