# 第10章配套笔记：图片输入与全连接参数详解

## 一、普通表格数据 vs 图片数据的输入区别

### 1) 普通表格数据（如 3->2->1 网络）
- 一个样本 = 一行数据
- 一个特征 = 一列数据
- 例子：身高、体重、年龄（3 个特征）
- 输入形状：`(3,)` 一维向量
- 输入层神经元数量 = 特征数量 = `3`

### 2) 图片数据（以 28×28 灰度图为例）
- 一个样本 = 一整张图片
- 一个特征 = 一个像素点
- 图片有空间结构（行列与邻接关系）
- 原始输入形状：`(28, 28)` 二维矩阵
- 送入全连接前需要展平：`28 * 28 = 784`
- 输入层神经元数量 = 总像素数 = `784`

---

## 二、灰度图片在深度学习中的数据表示
- 类型：灰度图（黑白）
- 尺寸：高 `28` × 宽 `28`
- 数据结构：二维数组（NumPy 矩阵）
- 每个像素取值：`0 ~ 255`
  - `0`：纯黑
  - `255`：纯白
  - 中间值：不同灰度

示例（简化 3×3）：

```text
[
  [  0,  56, 128],
  [ 23, 255,  90],
  [  0,  33, 180]
]
```

展平后：

```text
[0, 56, 128, 23, 255, 90, 0, 33, 180]
```

---

## 三、全连接层（Dense）的核心特点
1. 层层全连接：上一层每个神经元都连到下一层每个神经元。  
2. 无空间结构意识：输入展平后，行列关系丢失。  
3. 参数数量大：连线越多，权重越多，层一深就容易参数爆炸。  
4. 更偏全局统计：不擅长原生捕捉边缘/纹理等局部视觉模式。

---

## 四、全连接层权重、偏置、参数计算（实例）

以 `28×28` 灰度图 -> 隐藏层 `100` 神经元为例：

- 输入神经元数：`28 * 28 = 784`
- 隐藏层神经元数：`100`

### 1) 权重（Weight）数量
每条连线一个权重：

`weights = 784 * 100 = 78400`

### 2) 偏置（Bias）数量
每个隐藏神经元一个偏置：

`bias = 100`

### 3) 总参数（Parameters）

`params = weights + bias = 78400 + 100 = 78500`

---

## 五、最重要概念一句话总结
1. 权重：神经元之间连线的可学习系数。  
2. 偏置：每个神经元自己的常数项。  
3. 参数：训练时需要更新的变量总和 = 权重 + 偏置。  
4. 图片进全连接必须展平，空间结构会丢失，且参数容易爆炸，这就是 CNN 出现的核心动机之一。
