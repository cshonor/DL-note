# 卷积层 + Padding + Stride 完整新手笔记（含多通道卷积）

## 一、先统一符号（后面全部按这个来）
- 输入图大小：`H × W`（高 × 宽）
- 卷积核大小：`K × K`（这里用 `3×3`，即 `K=3`）
- Padding 填充圈数：`P`（常见 `P=1`，四周补一圈 0）
- Stride 步幅：`S`（常用 `S=1` 或 `S=2`）

---

## 二、Padding 填充：在哪一步做？做在哪里？

### 1) Padding 是什么
在原图四周补 0，让图变大一圈。

### 2) 在哪个步骤做
顺序固定：
1. 先对输入图做 padding（补 0）
2. 再在补完 0 的图上做卷积

也就是**卷积之前先填，不是卷积之后再填**。

### 3) 是不是只在第一层做 padding
不是。  
每进入一层卷积前，都可以独立选择 padding 或不 padding。

### 4) 5×5 图补 1 圈 0 的样子
原图（5×5）：

```text
a b c d e
f g h i j
k l m n o
p q r s t
u v w x y
```

补 1 圈 0 后（7×7）：

```text
0 0 0 0 0 0 0
0 a b c d e 0
0 f g h i j 0
0 k l m n o 0
0 p q r s t 0
0 u v w x y 0
0 0 0 0 0 0 0
```

---

## 三、Stride 步幅：卷积核怎么移动

### 1) Stride 的含义
- `Stride=1`：每次右/下移动 1 格
- `Stride=2`：每次右/下移动 2 格

Stride 越大，输出通常越小。

### 2) 常见使用
- 想尽量保尺寸：常见 `S=1`
- 想快速缩小：常见 `S=2`

---

## 四、输出大小公式（新手必背）

输出高度（宽度同理）：

`H_out = floor((H + 2P - K) / S) + 1`

---

## 五、标准示例：5×5 图 + 3×3 核

### 情况 A：`P=0, S=1`

`H_out = (5 + 0 - 3) / 1 + 1 = 3`

输出：`3×3`

### 情况 B：`P=1, S=1`

`H_out = (5 + 2 - 3) / 1 + 1 = 5`

输出：`5×5`（与输入同尺寸）

### 情况 C：`P=1, S=2`

`H_out = (5 + 2 - 3) / 2 + 1 = 3`

输出：`3×3`

---

## 六、输入 1 张图，输出 8 张特征图，下一层怎么处理？

这是最关键的多通道直觉：

### 1) 第一层卷积
- 输入：1 张灰度图（厚度=1）
- 使用 8 个卷积核
- 输出：8 张特征图（厚度=8）

### 2) 第二层卷积
此时输入变成“8 张图叠在一起”。

如果第二层设 10 个卷积核：
- 每个卷积核都要同时看这 8 张输入图
- 每张图各卷积出一个中间结果
- 把这 8 个中间结果相加，再加偏置
- 得到这个卷积核对应的 1 张输出图

10 个卷积核重复上面过程，最终输出就是 **10 张特征图**。

### 3) 关键结论
- 输入厚度：8
- 本层卷积核个数：10
- 输出厚度：10

**不是 8×10=80 张。**

---

## 七、一条完整流程线

### 输入
- `28×28` 灰度图，厚度=1

### 第一层卷积
- 3×3 核，数量=8
- `P=1, S=1`
- 输出：`28×28×8`

### 第二层卷积
- 3×3 核，数量=10
- 自动在 8 通道上卷积并相加
- `P=1, S=1`
- 输出：`28×28×10`

### 第三层卷积
- 3×3 核，数量=16
- 输出：`28×28×16`

### 接全连接分类
把 `28×28×16` 展平成一维向量，再送入全连接层。

---

## 八、新手终极总结
1. Padding 在卷积前做（先补 0 再卷积）。  
2. 每层卷积前都可独立选择是否 padding。  
3. Stride 决定每次滑几格。  
4. 输出尺寸看公式：`H_out = floor((H + 2P - K) / S) + 1`。  
5. 多通道卷积里，一个卷积核会把所有输入通道卷积后求和，得到一张输出图。  
6. 输出厚度永远等于本层卷积核数量。  
7. 多层卷积后，特征表达能力逐步增强，厚度由卷积核数量决定。
