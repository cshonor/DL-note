# 常用卷积核（特征提取）直观笔记（纯新手版）

> 目标：讲清“卷积核到底在提什么特征、权重长什么样、为什么能提出来”。

---

## 一、最核心结论
**卷积核 = 特征检测器。**  
**卷积核权重 = 它在找的图案模板。**

不同权重矩阵会对应不同响应：
- 竖线
- 横线
- 斜线
- 边缘/明暗交界
- 点、角点、纹理等

---

## 二、四个最常见 3×3 卷积核

## 1) 竖线检测核

```text
[
  [-1,  0,  1],
  [-1,  0,  1],
  [-1,  0,  1]
]
```

作用：检测左右方向的明暗突变，常用于竖直边缘响应。

---

## 2) 横线检测核

```text
[
  [-1, -1, -1],
  [ 0,  0,  0],
  [ 1,  1,  1]
]
```

作用：检测上下方向的明暗突变，常用于水平边缘响应。

---

## 3) 边缘增强/检测核（常见简化形式）

```text
[
  [ 0, -1,  0],
  [-1,  4, -1],
  [ 0, -1,  0]
]
```

作用：突出与周围差异大的区域（轮廓、细线、局部突变）。

---

## 4) 45 度斜线响应核（示意）

```text
[
  [-1, -1,  2],
  [-1,  2, -1],
  [ 2, -1, -1]
]
```

作用：更偏向响应某一斜向结构（方向相关检测）。

---

## 三、卷积核权重的直观规律

1. **正负号在做“亮暗对比”**  
   - 正权重区域：希望亮一点  
   - 负权重区域：希望暗一点  

2. **权重排列方向对应检测方向**  
   - 左右对比强 -> 竖边  
   - 上下对比强 -> 横边  
   - 对角对比强 -> 斜边  

3. **卷积输出大小反映匹配程度**  
   - 输出大：更像该核在找的模式  
   - 输出小或接近 0：不匹配

---

## 四、曲线、圆形、三角形怎么来？

单个 3×3 卷积核通常只能提取点、线、边等局部基础模式。  
复杂形状来自多层卷积逐层组合：

1. 第一层：点、线、边  
2. 第二层：角、弧、小部件  
3. 更深层：形状与语义部件  
4. 最后：完整目标

一句话：  
**单层看“线”，多层看“形”。**

---

## 五、卷积核是手写的吗？

通常不是手写。
- 初始化：随机
- 训练中：通过反向传播自动更新
- 训练后：学成对任务有用的特征模板

也就是：
**结构是人设计的，具体核权重是模型学出来的。**

---

## 六、新手终极总结
1. 卷积核是特征检测器，权重决定“检测偏好”。  
2. 3×3 常见核可响应竖线、横线、斜线和边缘。  
3. 正负权重和排列方向决定了响应类型。  
4. 复杂形状靠多层卷积逐步组合得到。  
5. 核参数通常由训练自动学习，不是人工固定。
