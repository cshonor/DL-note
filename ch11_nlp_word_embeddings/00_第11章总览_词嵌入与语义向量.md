# 第11章：能够理解自然语言的神经网络 —— 词嵌入与语义向量
**Core Concept: NLP Supervision + Embedding Space + Semantic Arithmetic**

## 本章整体定位
本章是从“数字/图像”跨到“文本语义”的关键转折章。  
核心任务：让你真正理解词向量（Embedding）如何承载语义关系。

---

## 章节核心学习目标
1. 理解 AI 处理语言到底在学什么  
2. 吃透词嵌入（Embedding）的核心机制  
3. 搭建简易文本分类网络  
4. 看懂语义如何映射到向量空间  
5. 理解“国王 - 男人 + 女人 ≈ 女王”的本质

---

## 小节主线导读

### 11.1 - 11.3 语言任务与监督学习
- 明确 NLP 目标：预测文本标签/关系，而不是“像人类一样理解”。  
- 输入文本 -> 模型预测 -> 损失计算 -> 反向更新，流程与其他任务一致。

### 11.4 - 11.6 IMDB 情感分类入门
- 通过标准数据集做“好评/差评”二分类。  
- 建立从文本到分类输出的最小可运行认知。

### 11.7 - 11.11 嵌入层与语义空间
- 嵌入层把离散词 ID 映射成连续向量。  
- 语义相近词在向量空间距离更近。  
- 网络神经元会对不同语义维度形成可分辨响应。

### 11.12 - 11.15 语言预测与类比
- 从完形填空到词类比，展示向量空间语义结构。  
- 经典示例：`king - man + woman ≈ queen`。

### 11.16 本章小结
- 完成“词 -> 向量 -> 语义关系 -> 预测任务”的闭环。

---

## 本章新手最终收获
- 理解大模型词向量机制的底层地基  
- 不再把语义建模当作黑箱魔法  
- 能独立搭出简单文本语义分类网络
