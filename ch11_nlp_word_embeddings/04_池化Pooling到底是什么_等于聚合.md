# 池化（Pooling）到底是什么：它其实就等于聚合

> 这篇只讲一个点：  
> **在你现在这条学习线里，池化 = 聚合。**

---

## 1. 一句话先讲透
你学的“聚合层”和教材里说的“池化层（Pooling）”，在这里本质是同一个动作：

**把一串词向量压缩成一个句子向量。**

- 口语说法：聚合
- 教材说法：池化

---

## 2. “池化”这个词到底在说什么
把一句话的所有词向量想成一池子数据：
- 词1向量
- 词2向量
- 词3向量
- ...

池化做的事就是：

**从这一池向量里提炼一个最能代表整句的向量。**

所以它就是“压缩 + 总结”。

---

## 3. 池化怎么做（你已学过的三种）
目标都一样：  
`(N, dim) -> (1, dim)`

只是规则不同：

### ① 平均池化（Mean Pooling）
每个维度把 N 个词向量对应数值求平均。

### ② 最大池化（Max Pooling）
每个维度只保留 N 个数里的最大值。

### ③ 求和池化（Sum Pooling）
每个维度把 N 个数直接相加。

---

## 4. 你必须记住的本质
**池化 = 数量压缩，不改单个向量维度。**

例如：
- 输入：5 个词，每词 100 维 -> `(5, 100)`
- 池化后：1 个句向量，仍是 100 维 -> `(1, 100)`

维度没变，条数变少。

---

## 5. 放回你当前 NLP 主线里
你现在的链路是：

1. 文本 -> 词 ID  
2. 嵌入层 -> 词向量序列（如 `5×100`）  
3. 池化/聚合 -> 句向量（`1×100`）  
4. 全连接层 -> 分类输出

没有池化时，全连接层拿到的是“多个词向量”；  
有池化后，它拿到的是“一个句子向量”。

---

## 6. 最短背诵版
- 池化 = 聚合  
- 池化把一串词向量提炼成一个句子向量  
- 最常用是平均池化  
- 形状核心变化：`(N, dim) -> (1, dim)`
