# 词袋、平均词向量、转移矩阵、句子向量：一口对齐

这篇用最直白的方式，把这四个概念和 RNN 串成一条线。

---

## 1. 词袋模型（Bag of Words）

最简单、最原始的文本表示，**完全不考虑顺序**。

例如句子：我、喜欢、编程。

假设词典：`{我:0, 喜欢:1, 编程:2, 你:3}`，词袋向量可以是：

```
[1, 1, 1, 0]
```

含义：出现过对应位置为 1，没出现为 0（也可用计数版词袋，这里用 0/1 举例）。

问题：

- **顺序完全没用**
- 「我打你」和「你打我」词袋可以一模一样

所以才需要 RNN 这类**带时序记忆**的模型。

---

## 2. 平均词向量（Average Word Embedding）

比词袋强一点，但**仍然不看顺序**。

做法：

1. 每个词先变成词向量（例如 128 维）
2. 把句子里所有词向量**相加**
3. **除以词数**，得到平均向量

例如 3 个词：

```
句子向量 = (vec1 + vec2 + vec3) / 3
```

优点：

- 不管句子多长，最后都是**固定维度的一个向量**
- 可以直接喂给全连接层

缺点：

- 不记顺序
- 信息被平均冲淡

这是 RNN 普及前很常见的「笨办法」，和现在学的 RNN 句子表示可以对照着记。

---

## 3. 转移矩阵（Transition Matrix）

在基础 RNN 里，**转移矩阵就是循环权重 `W`**。

它只做一件事：

**把上一步的隐藏状态，变成这一步计算里「来自过去」的那一块。**

递推式：

```
h_t = tanh( W @ h_(t-1) + U @ x_t )
```

- **`W`**：转移矩阵，管**记忆怎么从过去传到当前**
- **`U`**：输入矩阵，管**当前输入 `x_t` 怎么进隐藏空间**

「转移」的意思：把**过去的记忆**变换、融合进**当前这一步**的计算里。

所以：**转移矩阵 = 控制记忆沿时间如何流动的矩阵**（在 RNN 里就是 `W`）。

---

## 4. 句子向量（Sentence Vector）怎么来

**句子向量 = 把整句话压成一个固定维度的向量**，用来做分类、检索、下游全连接等。

常见三种构建方式：

### ① 平均词向量（最简单）

所有词向量相加再除以词数（见上文第 2 节）。

### ② RNN 最后一步隐藏状态（many-to-one 里很标准）

- 词 1 得到 `h_1`
- 词 2 得到 `h_2`
- …
- 最后一个词得到 `h_T`

在「整句出一个结果」的设定里，常把 **`h_T` 当作句子级的记忆总结**（再经输出层 `V @ h_T + b` 得到最终预测，见 `04` 与 `01` 里的误区说明）。

### ③ RNN 各步隐藏状态再平均

把 `h_1 … h_T` 全部加起来再平均，有时更稳，但信息会更糊一点。

---

## 四者关系（一句话各记一条）

| 概念 | 一句话 |
| --- | --- |
| 词袋 | 不看顺序，只看「有没有 / 出现几次」 |
| 平均词向量 | 不看顺序，把词向量揉成一个向量 |
| 转移矩阵 `W` | RNN 里把上一步记忆传到当前步 |
| 句子向量 | 整句的固定维表示；many-to-one 时常用 `h_T` 或再平均各步 `h` |

---

## 和你当前 RNN 知识块的衔接

你已经对齐了：

- RNN 循环、隐藏状态 `h` 即记忆
- many-to-one 时最后常用 `h_T` 作全局表示
- `U`、`W` 分工与误差回传（见 `03`）

接下来把 **BPTT（按时间反向传播）** 和 **梯度消失 / 爆炸** 吃透，基础 RNN 这条线就闭环了，再自然过渡到 LSTM。
