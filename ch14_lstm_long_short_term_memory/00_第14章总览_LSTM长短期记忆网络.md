# 第14章：LSTM 长短期记忆网络总览
**Core Concept: Gated Memory for Long Sequences**

## 本章整体定位
第14章解决一个核心问题：  
为什么普通 RNN 在长序列上容易遗忘，以及 LSTM 如何用门控机制稳定保留长期信息。

它是从基础 RNN 过渡到更强序列模型（GRU/Transformer）的关键桥梁。

---

## 章节核心学习目标
1. 看懂普通 RNN 的长程记忆瓶颈  
2. 理解 LSTM 的细胞状态 `C` 与隐藏状态 `h` 分工  
3. 吃透遗忘门、输入门、输出门三门协作逻辑  
4. 能写出 LSTM 单步前向公式并解释每一项含义  
5. 能在 PyTorch 中完成最小 LSTM 文本建模流程
