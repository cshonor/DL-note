# LSTM 按 I/H/C 符号：门如何由输入与 H 共同计算

这篇只讲一件事：  
**`I_t` 和 `H_(t-1)` 到底如何一起算出三门和候选记忆，再更新 `C_t`。**

---

## 1) 先记总论

在每个时间步，下面四个量都来自同一份输入原料：
- `f_t`（遗忘门）
- `i_t`（输入门）
- `o_t`（输出门）
- `C_t_tilde`（候选新记忆）

这个输入原料就是：

`concat_t = [I_t; H_(t-1)]`

---

## 2) 第一步：拼接输入与上一步输出

每个时间步有三样输入状态：
- `I_t`：当前输入
- `H_(t-1)`：上一步外部循环状态
- `C_(t-1)`：上一步内部长期记忆

其中 `I_t` 和 `H_(t-1)` 先拼接：

`concat_t = [I_t; H_(t-1)]`

---

## 3) 第二步：同一份 concat 计算四个结果

### 遗忘门
`f_t = sigmoid(W_f @ concat_t + b_f)`

### 输入门
`i_t = sigmoid(W_i @ concat_t + b_i)`

### 输出门
`o_t = sigmoid(W_o @ concat_t + b_o)`

### 候选新记忆
`C_t_tilde = tanh(W_c @ concat_t + b_c)`

直观分工：
- `f_t, i_t, o_t` 是 0 到 1 的门控比例
- `C_t_tilde` 是当前时刻可写入的候选内容

---

## 4) 第三步：更新内部长期记忆 C

`C_t = f_t * C_(t-1) + i_t * C_t_tilde`

对应含义：
- `f_t * C_(t-1)`：保留多少旧记忆
- `i_t * C_t_tilde`：写入多少新记忆

两项相加得到新的 `C_t`。

---

## 5) 第四步：生成外部循环状态 H

`H_t = o_t * tanh(C_t)`

这一步把内部记忆 `C_t` 经过门控暴露成 `H_t`。  
`H_t` 才是下一时间步继续循环使用的外部状态。

---

## 6) 一条完整计算链路（第 t 步）

1. 读入：`I_t`, `H_(t-1)`, `C_(t-1)`  
2. 拼接：`concat_t = [I_t; H_(t-1)]`  
3. 计算：`f_t`, `i_t`, `o_t`, `C_t_tilde`  
4. 更新：`C_t = f_t * C_(t-1) + i_t * C_t_tilde`  
5. 输出：`H_t = o_t * tanh(C_t)`  
6. 传递：`H_t`, `C_t` 进入下一步

---

## 最简背诵版

`I` 和 `H` 先拼接，拼接向量一次算出三门加候选记忆；  
再用 `f/i` 更新 `C`，最后用 `o` 从 `C` 生成 `H` 并继续循环。
